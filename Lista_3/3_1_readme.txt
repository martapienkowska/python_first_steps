Task: Write module which convert Fahrenheit - Celsius, Celsius - Fahrenheit.
Save file with random Celsius degrees and convert them to Fahrenheit. Check if your calculations are correct.

Information about files:
- module_3_1 - contains functions which calculate Fahrenheit to Celsius, Celsius to Fahrenheit and which create random Celsius degrees.
- 3_1_file_write - writes random Celsius degrees in celcjusz.txt file
- 3_1_file_read - reads celcjusz.txt, convert Celsius to Fahrenheit and write Fahrenheit in fahrenheit.txt
- 3_1_file_verification - check if calculations from file_read are correct
